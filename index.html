⚙️ Google Sheets 연동 설정

1
Google Sheets 만들기
sheets.new 접속 → 새 스프레드시트 생성 (구글 로그인 필요, 최초 1회만)
2
Apps Script 코드 붙여넣기
시트 상단 → 확장 프로그램 → Apps Script
기존 코드를 전부 지우고 아래 코드를 붙여넣은 후 저장(💾)
3
웹 앱으로 배포
Apps Script 상단 → 배포 → 새 배포
· 유형 선택: 웹 앱
· 다음 사용자로 실행: 나
· 액세스 권한: 모든 사용자 ← 로그인 불필요하게 하는 옵션
→ 배포 클릭 → 구글 권한 허용 → 웹 앱 URL 복사
4
URL 입력 후 저장
